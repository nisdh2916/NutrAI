# NutrAI 도메인 용어집

## 사용자 (User)
건강 관리 또는 체중 조절이 목표인 일반 성인. 당뇨·고혈압 등 특정 건강 상태를 가진 사람도 포함.

## 끼니 (Meal)
사용자가 하루 중 섭취하는 단위 식사. **5종**으로 분류:

| 종류 | 설명 |
|------|------|
| 아침 | 오전 식사 |
| 점심 | 낮 식사 |
| 간식 | 오후 가벼운 섭취 |
| 저녁 | 저녁 식사 |
| 야식 | 늦은 밤 섭취 |

음식 추가 시 시간대 기반으로 끼니를 자동 추천하되, 사용자가 직접 변경할 수 있다.

## 음식 (MealFood)
끼니를 구성하는 개별 식품. 이름·칼로리·탄수화물·단백질·지방 정보를 포함.

## 식단 기록 (MealRecord)
특정 끼니에 섭취한 음식들의 묶음. 끼니 종류와 시각을 포함.

## 건강 목표 (Goal)
사용자가 설정한 식단 관리 방향. `다이어트`, `체중 유지`, `근육 증진`, `건강 관리` 4종.

## 알레르기 (Allergy)
사용자가 기피해야 할 식품 성분. 11개 카테고리: 유제품, 견과류, 갑각류, 밀, 글루텐, 계란, 대두, 복숭아, 토마토, 고등어, 조개류.

## 건강 상태 (Condition)
사용자의 기저 질환 또는 건강 상태 (예: 당뇨, 고혈압). AI 추천 시 제약 조건으로 활용.
