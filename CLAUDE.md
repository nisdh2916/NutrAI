# NutrAI 프로젝트 지시사항

## 버그 수정 후 문서화 규칙

버그를 수정하거나 문제를 해결한 세션에서는 반드시 `docs/troubleshooting.md`를 업데이트하세요.

형식:
```
## N. [제목]

**증상:** 어떤 문제가 발생했는지

**원인:** 왜 발생했는지

**해결:**
\```python  (또는 dart, bash 등)
# 변경 전
...

# 변경 후
...
\```

**관련 파일:** `경로/파일명.py` → `함수명()`
```

규칙:
- 새 항목은 기존 번호에 이어서 추가 (현재 10번까지 있음)
- 코드 변경이 있었을 때만 추가 (단순 질의응답은 제외)
- 같은 문제의 재발이면 기존 항목 업데이트
